# はじめに

![](/images/demo.jpeg)

こんな感じの

- 認証
- 画像投稿
- 記事投稿/編集

を備えたデモアプリを

- フレームワーク: Next.js 14
  - App Router
  - Server Actions
  - RSC(React Server Components)
- UI: shadcn/ui / Tailwind CSS
- 認証: Clerk
- ORM: Prisma
- DB: Vercel Postgres
- Storage: Cloudflare R2
- Hosting: Vercel

を使って実装するガイドです。スタイルは省略し、機能的な部分に焦点を絞っています。

[デモ](https://next-demo-app-henna.vercel.app) / [ソースコード](https://t.co/r2MCRW15YU)
